# reqtxtupd8
[![PyPI version](https://badge.fury.io/py/reqtxtupd8.svg)](https://badge.fury.io/py/reqtxtupd8)
> Writes your python package list to requirements.txt


## Install
Install with pip
```sh
$ pip install reqtxtupd8
```

## Run
Execute in any directory
```sh
$ requpd8
```
